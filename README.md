# Ioasys Book
Aplicativo para listagem de Livros.

## Bibliotecas utilzadas ##

 - React
 - Typescript
 - Axios 
 - Redux
 - Styled Components
 - React Navigation
 - React Native Custom Statusbar



## Instalação ##
`yarn or npm install`

`cd ios && pod install`


## Uso ##

`yarn android or yarn ios`
